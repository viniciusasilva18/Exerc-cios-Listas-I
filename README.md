## O que tem

### lista_convidados.py
Cadastra 5 convidados e mostra a lista.

### controle_precos.py
Guarda 5 preços e mostra o maior e menor.

### lista_tarefas.py
Adiciona tarefas até digitar `fim`.

### controle_temperatura.py
Converte Celsius para Fahrenheit e mostra as médias.

### lista_compras.py
Menu simples pra:
- adicionar
- pesquisar
- remover
- alterar
- listar produtos
